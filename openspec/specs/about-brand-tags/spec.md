# Spec: 品牌标签与 Logo 占位

## 概述

「关于我」区块**下方**展示品牌相关标签/合作方列表；每项含名称与可选外链。正式 Logo 未就绪时，以 `design.md` 约定的**灰底圆角矩形 + 缩写文字**作为占位，不依赖外链占位图服务。

## 行为规格

### 场景：品牌区展示

**Given** 用户已将视口滚动至 `#about` 或从导航跳转至该 section  
**When** 用户查看区块底部品牌区域  
**Then** 可见至少一个品牌项；每项可见名称（及占位方块内的缩写）；布局在 `md` 上下均不换行溢出整页（允许标签换行或栅格折行）

### 场景：带外链的品牌项

**Given** 某品牌项在 `content` 中配置了 `href`  
**When** 用户点击该项链接  
**Then** 在新标签页打开时使用 `rel="noopener noreferrer"`（或与项目卡片 GitHub 链接相同的安全策略）

## 边界与异常

### 场景：品牌列表为空（数据层）

**Given** `content` 中品牌数组被误配置为空数组  
**When** 页面渲染「关于我」  
**Then** 不展示空壳栅格误导用户，或展示开发期断言/构建失败（以 `tasks` 实施约定为准）；**不得**渲染仅含空白占位方块的无限拉伸区域
