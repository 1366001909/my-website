# Spec: Hero 首屏信息展示

## 概述

首屏提供全屏高度（`100svh`）的 Hero 区域，在视口内居中展示姓名、职业与一句话介绍；文案在常见桌面与移动宽度下可读、不溢出。

## 行为规格

### 场景：首屏展示核心信息

**Given** 用户已打开站点首页且首屏内容已加载  
**When** 用户查看首屏 Hero 区域  
**Then** 用户可见主标题（姓名）、职业与一句话介绍，且三者视觉层级清晰（姓名为最高层级）

### 场景：首屏占满视口高度

**Given** 用户设备浏览器支持 `svh` 单位  
**When** 用户查看首屏 Hero 区域  
**Then** Hero 区域最小高度为 `100svh`，且核心文案在视口内垂直与水平居中

### 场景：无 JavaScript 时的降级

**Given** 页面在 JavaScript 不可用或尚未执行完成的情况下仍可渲染静态 HTML/CSS  
**When** 用户查看首屏  
**Then** 用户仍可见姓名、职业与一句话介绍（不要求 Canvas 粒子层可用）

## 边界与异常

### 场景：窄屏与较长文案

**Given** 视口宽度小于常见桌面断点（例如移动设备竖屏）且姓名或一句话介绍文本较长  
**When** 用户查看首屏 Hero  
**Then** 文案在容器内换行或收缩排版，不出现横向滚动条遮挡阅读，且 CTA/主题控件（若与 Hero 同区）不与正文严重重叠
