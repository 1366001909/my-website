# Spec: 项目卡片悬浮微动效

## 概述

在指针设备上，用户将鼠标悬浮于单张项目卡片上时，提供**轻微**视觉反馈（如阴影加深、微缩放、边框高亮等），以增强可交互感知。须尊重系统「减少动效」偏好。

## 行为规格

### 场景：指针悬浮于卡片

**Given** 用户设备未开启 `prefers-reduced-motion: reduce`，且使用指针设备  
**When** 用户将鼠标指针移入某项目卡片区域  
**Then** 该卡片相对默认状态出现轻微视觉变化（具体组合以 `design.md` 为准），变化过渡时长短促、不干扰阅读

### 场景：指针移出卡片

**Given** 用户已将指针移入某卡片并触发悬浮态  
**When** 用户将指针移出该卡片  
**Then** 卡片恢复至默认视觉状态

## 边界与异常

### 场景：用户偏好减少动效

**Given** 用户系统开启 `prefers-reduced-motion: reduce`  
**When** 用户将指针移入项目卡片  
**Then** 不出现明显位移动画或可关闭位移动画；仅允许极弱或无过渡的静态高亮（以 `design.md` 为准），且不得依赖长时长 CSS animation 循环

### 场景：仅键盘或触屏无悬浮

**Given** 用户仅使用键盘 Tab 或触屏设备无「悬浮」语义  
**When** 用户聚焦或点按卡片  
**Then** 卡片仍具备 `:focus-visible` 或等效可辨识焦点样式；不得将唯一信息隐藏在仅 `:hover` 才出现的状态中
