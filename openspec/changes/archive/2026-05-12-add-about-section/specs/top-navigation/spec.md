# Spec: 顶部导航栏 — 关于我入口（增量）

## 概述

在既有固定顶栏基础上，主导航增加「关于我」链至 `#about`，链接顺序为 **首页 → 项目 → 关于我 → 联系我**；不改变「无搜索、无下拉、无登录」约束。

## 行为规格

### 场景：右侧主导航链接数量与顺序

**Given** 导航栏已渲染  
**When** 用户查看右侧一级链接  
**Then** 可见四个链接，从左到右依次为：首页、项目、关于我、联系我；「关于我」的 `href` 为 `#about`（或通过等效程序化导航与 `href` 一致）

### 场景：激活态与无障碍

**Given** 当前滚动上下文判定为「关于我」section 为主可见（与现有 `IntersectionObserver` 策略一致）  
**When** 用户查看导航栏  
**Then**「关于我」锚点具备与其他链接一致的 `aria-current="page"` 规则（仅当前 section 对应项为 `page`）

## 边界与异常

### 场景：窄屏四链布局

**Given** 视口宽度小于 `md`  
**When** 用户查看导航栏  
**Then** 四个链接仍可访问（允许换行或压缩间距，与现有顶栏窄屏策略一致）；不得为此引入汉堡菜单或多级菜单
