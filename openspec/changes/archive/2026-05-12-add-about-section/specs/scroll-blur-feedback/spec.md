# Spec: 滚动模糊反馈 — 纳入「关于我」导航（增量）

## 概述

用户通过主导航触发锚点跳转时的短时背景模糊反馈，**须**在用户点击「关于我」并触发与 `beginNavigate('about')` 等效滚动时，与点击「首页 / 项目 / 联系我」表现一致（含减少动效与快速连点约束）。

## 行为规格

### 场景：点击「关于我」触发模糊

**Given** 用户未开启 `prefers-reduced-motion: reduce`  
**When** 用户点击主导航中的「关于我」  
**Then** 在滚动过渡期间可见与点击其他主导航链一致的模糊反馈层；滚动结束后层消失且不阻挡指针事件

## 边界与异常

### 场景：减少动效下点击「关于我」

**Given** 用户系统为 `prefers-reduced-motion: reduce`  
**When** 用户点击「关于我」  
**Then** 模糊反馈遵循 `design.md` 与现有实现中对动效降级的同一策略（关闭或极短低强度），不得单独为「关于我」恢复长时长高模糊
